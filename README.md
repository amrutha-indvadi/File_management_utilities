# Windows File Utilities
A C application that demonstrates various Windows file management operations including disk space monitoring, ZIP archive creation, archive extraction, and MD5 checksum generation.

## Features
* Retrieve disk space information
* Compress folders into ZIP archives
* Extract ZIP archives
* Generate MD5 checksums for archive verification
* Execute Windows system utilities through C

## Utilities Included

### Disk Space Monitor
Displays:
* Total disk space
* Available free space
* Free space accessible to the current user

### ZIP Archive Management
* Compress folders into ZIP files using PowerShell
* Extract ZIP archives to a specified location

### MD5 Checksum Generation
* Generate MD5 hash values for archive integrity verification using Windows CertUtil
